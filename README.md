## Projeto de Bancos de Dados 

- Compreender conceitos relacionados ao armazenamento, manipulação e resgate de dados em banco de dados e arquivos para entendimento dos processos de forma eficiente;
- Aplicar os diferentes tipos de modelos na construção de banco de dados para um mapeamento do banco de dados de forma coerente;
- Criar consultas de banco de dados para o resgate de dados de maneira precisa;
- Compreender conceitos referentes ao relacionamento entre entidades em banco de dados para desenvolvimento de maneira lógica.
